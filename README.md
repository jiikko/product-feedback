# Product Feedback

このリポジトリは、各プロダクトへのお問い合わせ・不具合報告・機能要望の受付窓口です。

## 対象プロダクト

- BabyNote
- SnapTrim
- ThumbnailThumb
- Multi-Video Player for Dropbox

## お問い合わせ方法

`Issues` の `New issue` から、目的に合ったフォームを選んで投稿してください。

- 不具合報告（Bug）
- 機能要望（Feature）
- お問い合わせ（Support）

## 記入してほしい内容

- 対象プロダクト
- アプリバージョン
- OSバージョン / 端末名
- 再現手順
- 期待した結果
- 実際の結果
- （可能なら）スクリーンショットや動画

## 注意事項

個人情報・機密情報（メールアドレス、電話番号、アカウント情報など）は投稿しないでください。

## 返信ポリシー

すべての投稿は確認しますが、すべてに返信できるとは限りません。  
対応優先度は、通常「不具合修正」を「機能要望」より優先します。

## セキュリティに関する報告

脆弱性などセキュリティ問題は Issue に公開せず、以下へご連絡ください。  
`support@example.com`（必要に応じて実際の連絡先に変更してください）
